### Utiliser Vos Photos

**Conseil principal:** Vos propres photos sont BIEN MIEUX!

Pour ajouter vos photos:

1. **Prendre les photos** avec votre smartphone
   - Bon éclairage (jour de préférence)
   - Plusieurs angles
   - Gros plans sur les détails

2. **Télécharger sur Imgur.com (gratuit)**
   - Allez sur https://imgur.com
   - Cliquez "Upload images"
   - Sélectionnez votre photo
   - Copiez le lien direct
   - Collez dans le formulaire admin

3. **Alternative: Imgbb.com**
   - Plus facile, interface en français
   - https://imgbb.com
   - Upload, copier le lien, c'est tout!

## Tips pour les Photos Professionnelles

### Éclairage
- Prenez en jour naturel si possible
- Évitez les contre-jours
- Pas d'ombres trop marquées

### Composition
- Centrez l'élément principal
- Incluez du contexte (la pièce)
- Plusieurs angles pour la même réalisation

### Taille
- Réduisez à 500-800px de largeur max
- Compressez à moins de 500KB par image
- Format: JPG ou PNG

### Nombre de photos
- Minimum 2 images par projet
- Idéalement 3-4 pour plus de détails
- Une image générale + gros plans

## Template Complet pour Ajouter un Projet

Copiez/collez dans l'admin:

```
TITRE: [Titre descriptif]

DESCRIPTION: [Texte de 150-250 mots expliquant le projet]

CATÉGORIE: [Portes EI30 / Cuisines / Chantiers]

IMAGES: [URL1], [URL2], [URL3]
```

### Exemple Copié/Collé:

```
TITRE: Cuisine Moderne Blanc et Bois

DESCRIPTION: Cuisine entièrement sur-mesure avec façades blanches laquées et plan de travail en chêne massif. Îlot central avec rangements intégrés. Électroménagers haut de gamme. Projet réalisé en 6 semaines. Client très satisfait du résultat final et de la qualité des finitions.

CATÉGORIE: Cuisines

IMAGES: https://images.unsplash.com/photo-1556909114-f6e7ad7d3136?w=500, https://images.unsplash.com/photo-1446776653964-20c1d3a81b06?w=500
```

## Fréquence de Publication

**Recommandation:**
- Publiez **au moins 1 nouveau projet par mois**
- Idéalement **2-3 par mois** pour montrer l'activité
- Variez les catégories (portes, cuisines, chantiers)

---

## Check-List Avant de Publier

Avant de cliquer "Enregistrer":

- [ ] Le titre est court et descriptif (< 80 caractères)
- [ ] La description est complète (150-300 mots)
- [ ] La catégorie est correcte
- [ ] Les images s'affichent (testez les URLs)
- [ ] Minimum 2 images par projet
- [ ] Les images sont de bonne qualité
- [ ] Le projet correspond à vos réalisations réelles
