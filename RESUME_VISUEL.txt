┌─────────────────────────────────────────────────────────────────────────────┐
│                      GUIDE RAPIDE 5 ÉTAPES                                  │
└─────────────────────────────────────────────────────────────────────────────┘

1️⃣  INSTALLATION (1 minute)
    $ npm install

2️⃣  CONFIGURATION (30 secondes)
    - Créez/modifiez .env
    - Changez mot de passe admin

3️⃣  DÉMARRAGE (30 secondes)
    $ npm start
    Serveur tourne sur http://localhost:3000

4️⃣  ACCÈS (1 minute)
    - Site public: http://localhost:3000
    - Admin: Clic 3x sur 🔐 en bas du site

5️⃣  REMPLISSAGE (15 minutes+)
    - Ajoutez vos projets
    - Modifiez les paramètres
    - Testez les formulaires

✨ C'est fait! Votre site fonctionne!
