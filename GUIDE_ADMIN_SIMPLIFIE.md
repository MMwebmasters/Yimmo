## 🏗️ Ajouter un Projet

### Comment ajouter une réalisation?

1. **Cliquez sur "🏗️ Projets"** dans le menu de gauche
2. Cliquez sur le bouton **"+ Ajouter un projet"** (haut droite)
3. Remplissez le formulaire:

   **Titre du projet** (obligatoire)
   - Exemple: "Porte EI30 - Immeuble Centre Ville"

   **Description** (obligatoire)
   - Exemple: "Installation d'une porte coupe-feu EI30 certifiée dans un immeuble résidentiel. Design moderne, sécurité maximale."
   - Soyez détaillé mais concis (200-300 mots)

   **Catégorie** (obligatoire)
   - Choisissez entre:
     - 🚪 **Portes EI30**
     - 🍽️ **Cuisines**
     - 🏗️ **Chantiers**

   **Images** (optionnel mais recommandé)
   - Collez les URL des images, séparées par des virgules
   - Exemple: `https://example.com/image1.jpg, https://example.com/image2.jpg`
   - **Conseil:** Utilisez au moins 2-3 images par projet

4. Cliquez **"Enregistrer"**

✅ Le projet est maintenant visible sur le site public!

### Où trouver des images gratuits?
- **Unsplash.com** - Search: "kitchen", "door", "construction"
- **Pexels.com** - Search: "interior design"
- **Pixabay.com** - Complètement gratuit

**Astuce:** Vos propres photos sont les meilleures! Utilisez votre smartphone.

## 💬 Messages de Contact

### Voir les messages reçus

1. Cliquez sur **"💬 Messages"** dans le menu
2. Vous voyez tous les messages reçus

### Information par message
- **Nom** du visiteur
- **Email** et téléphone
- **Sujet** du message
- **Date** et heure
- **Statut:** "Nouveau" ou "Lu"

### Actions possibles

**Marquer comme lu**
- Cliquez "Marquer comme lu" pour noter que vous avez traité le message

**Copier l'email**
- Cliquez "Copier email" pour copier l'adresse email
- Vous pouvez ensuite la coller dans votre client email

**Répondre?**
- Copiez l'email et répondez depuis votre email personnel
- OU utilisez le bouton WhatsApp pour contacter directement

## 📅 Rendez-vous

## ❓ Problèmes Courants

### "Je n'arrive pas à accéder à l'admin"
- Vérifiez que vous cliquez 3 fois rapidement sur le cadenas 🔐
- Assurez-vous que le footer est visible (scrollez en bas)
- Rafraîchissez la page (Ctrl+R)

### "Mon projet n'apparaît pas sur le site"
- Allez vérifier que vous avez bien cliqué "Enregistrer"
- Rafraîchissez le site public (Ctrl+R)
- Vérifiez que vous avez rempli le titre ET la description

### "Je ne reçois pas les messages de contact"
- Vérifiez la section "💬 Messages"
- Les messages s'y affichent instantanément
- Vérifiez votre email de contact dans les paramètres

### "Les images ne s'affichent pas"
- Vérifiez que l'URL de l'image est valide
- Essayez l'URL dans le navigateur
- L'image doit être en format JPG, PNG, ou WebP

### "Je m'agrandis pas mes images"
- Pour ajouter une image, vous devez d'abord la mettre en ligne
- Utilisez un service gratuit comme:
  - **Imgur.com** - Hébergement d'images
  - **Imgbb.com** - Serveur d'images gratuit
  - **Unsplash.com** - Images libres
- Puis collez l'URL dans le formulaire
