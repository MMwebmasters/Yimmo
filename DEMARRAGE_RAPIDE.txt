🔧 ÉTAPE 1: Installation (2 minutes)

1. Ouvrez un terminal (Windows PowerShell ou CMD)

2. Allez au dossier du projet:
   cd "c:\Users\moham\OneDrive\Bureau\Web Vitrine\yimmo"

3. Installez les dépendances:
   npm install

   (Attendez que ça se termine...)

═══════════════════════════════════════════════════════════════════════════════════

⚙️ ÉTAPE 2: Configuration (1 minute)

1. Dans le dossier "yimmo", créez/modifiez le fichier ".env"

2. Mettez ceci dedans (ou modifiez le .env.example existant):

   PORT=3000
   NODE_ENV=development
   JWT_SECRET=abcd1234efgh5678ijkl9012mnop3456qrst7890
   ADMIN_EMAIL=Yousri@yimmosarl.com
   ADMIN_PASSWORD=votremotdepasseforticiii123

   (Changez "votremotdepasseforticiii123" par votre mot de passe)

═══════════════════════════════════════════════════════════════════════════════════

🚀 ÉTAPE 3: Démarrage (30 secondes)

1. Dans le terminal, tapez:
   npm start

2. Vous devriez voir:
   "✓ Serveur Yimmo démarré sur http://localhost:3000"
   "✓ Base de données connectée"
   "✓ Compte admin créé"

3. Le serveur est maintenant en ligne!

═══════════════════════════════════════════════════════════════════════════════════

🌐 ÉTAPE 4: Ouvrir le site (30 secondes)

1. Ouvrez votre navigateur web (Firefox, Chrome, Edge, Safari)

2. Tapez dans la barre d'adresse:
   http://localhost:3000

3. TADA! Le site Yimmo s'affiche!

═══════════════════════════════════════════════════════════════════════════════════

🔐 ÉTAPE 5: Accéder à l'Admin (2 minutes)

1. Sur le site public (localhost:3000)

2. Allez tout en bas de la page (scrollez)

3. Regardez le footer

4. Passez la souris sur le footer
   → Un petit cadenas 🔐 apparaît en bas à droite

5. CLIQUEZ 3 FOIS RAPIDEMENT sur le cadenas
   (Vite vite! 1, 2, 3!)

6. Une fenêtre de connexion s'ouvre

7. Connectez-vous:
   Email: Yousri@yimmosarl.com
   Mot de passe: (celui que vous avez mis dans .env)

8. Cliquez "Connexion"

9. BRAVO! Vous êtes dans l'admin!

💡 ASTUCES

• Mot de passe oublié? Modifiez le .env et redémarrez npm
• Le site ne se met pas à jour? Rafraîchissez (Ctrl+R)
• Erreur de port? Un autre serveur utilise 3000
  → Changez PORT dans .env (3001, 3002, etc.)
• Terminal figé? Ctrl+C pour arrêter, puis npm start pour redémarrer
